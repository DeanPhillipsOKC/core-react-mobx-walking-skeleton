# Clean Architecture Skeleton

A clean, bare bones application using .NET Core, React, and MobX

## Prerequisites

1. Dotnet Core SDK v3.1.x
2. NodeJS

## Run (Local)

1. From the API folder run `dotnet watch run`
2. From the client-app folder, run `npm start`
